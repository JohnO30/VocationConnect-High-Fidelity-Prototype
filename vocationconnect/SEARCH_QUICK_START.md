# Alumni Search Engine - Quick Start Guide

## Landing on the Search Page

Navigate to: `/alumni/search`

You'll see a clean, organized form with:
- **Search Box**: Main keyword input
- **Advanced Filters**: Hidden section with additional criteria
- **Sort Options**: How to order your results

## Basic Search (Keyword Only)

### Example 1: Find someone by name
1. Type `John Smith` in the search box
2. Click **Search Alumni** button
3. Get all alumni with "John" in first name or "Smith" in last name

### Example 2: Find alumni from a specific company
1. Type `Google` in the search box
2. Click **Search Alumni**
3. Get all alumni who work at Google

### Example 3: Find alumni with specific skills
1. Type `Python` in the search box
2. Click **Search Alumni**
3. Get all alumni whose skills include Python

## Using Advanced Filters

### Scenario 1: Find Tech Industry Mentors
1. Type `Machine Learning` in keyword search (optional)
2. Select **Technology** from Industry dropdown
3. Leave other filters as default
4. Click **Search Alumni**
5. Get technology professionals, optionally matching your keyword

### Scenario 2: Find Recently Graduated Alumni
1. Set **Minimum Graduation Year** to 2022
2. Leave keyword empty
3. Keep industry as "All Industries"
4. Click **Search Alumni**
5. Get all alumni who graduated in 2022 or later

### Scenario 3: Find Experienced Mentors
1. Set **Minimum Years of Experience** to 10
2. Set **Maximum Years of Experience** to 30
3. Select an industry (e.g., **Finance**)
4. Click **Search Alumni**
5. Get finance professionals with 10-30 years of experience

### Scenario 4: Find Available Interview Mentors
1. Check **Available for Mock Interviews**
2. Optionally select an industry
3. Click **Search Alumni**
4. Get alumni specifically available for interviews

### Scenario 5: Find Mentors from Your Graduation Year
1. Set **Minimum Graduation Year** to 2020
2. Set **Maximum Graduation Year** to 2020
3. Optionally add industry filter
4. Click **Search Alumni**
5. Get same-year graduates

## Real-Time Search Suggestions

### How to Use Autocomplete

1. Start typing in the search box (minimum 2 characters)
2. A dropdown appears with suggestions
3. Each suggestion shows:
   - **Name** (in bold)
   - **Company** (with 🏢 icon)
   - **Job Title**
   - **Industry**
4. Click any suggestion to auto-fill the search box
5. Click outside or wait for suggestions to disappear
6. Submit the form to search

### Example with Autocomplete
1. Type `John` → See suggestions like:
   - John Smith at Google · Software Engineer · Technology
   - John Davis at McKinsey · Consultant · Consulting
   - John Chen at Massachusetts Institute of Technology · Professor · Education
2. Click on "John Smith at Google"
3. Search box auto-fills with this full name
4. Click Search Alumni to find them

## Sorting Your Results

### After Getting Results

You'll see a **Sort by** dropdown on the results page:

**Options:**
- **Name (A-Z)**: Alphabetical by first name (default)
- **Experience (Most to Least)**: Find the most experienced mentors first
- **Graduation Year (Most Recent)**: Find recent graduates
- **Company (A-Z)**: Alphabetical by company

### Example: Change Sort
1. Get search results
2. Click **Sort by** dropdown
3. Select **Experience (Most to Least)**
4. Page instantly resorts without losing your search filters
5. Now see most experienced professionals first

## Understanding the Results

### Each Alumni Card Shows

- **Avatar Icon**: 👤 (personal placeholder)
- **Name**: Full name in bold
- **Company**: Current employer highlighted in blue
- **Job Title**: Professional role
- **Stats**:
  - 🏢 Industry
  - ⏳ Years of Experience
  - 🎓 Graduation Year
- **Skills**: Top 3 skills shown as tags, with "+N more" if additional skills
- **Badge**: Green badge if "Available for Mock Interviews"
- **Buttons**:
  - **View Full Profile →**: See complete profile details
  - **💬 Request Connection**: Send them a message (students only)

### Filter Summary

At the top of results, see **Active Filters** showing:
- 📝 Keyword search term
- 🏢 Selected industry
- ⏳ Experience range
- 🎓 Graduation year range
- 💬 Interview availability status

Click **← New Search** to modify filters

## Tips & Tricks

### When You Get Too Many Results
✓ Add a keyword (name, company, skill)
✓ Filter by industry
✓ Narrow the experience range
✓ Restrict graduation year range

### When You Get No Results
✓ Check spelling of keywords
✓ Remove some filters
✓ Broaden the experience range
✓ Widen graduation year range
✓ Try browsing all alumni instead

### Best Search Strategies
✓ Start specific, then broaden if needed
✓ Use industry filter to narrow large result sets
✓ Combine keyword + industry for best results
✓ Use experience level to find role-appropriate mentors
✓ Filter for interview availability if seeking mock interviews

### Mobile Tips
✓ Use industry dropdown to reduce typing
✓ Test autocomplete for quick name entry
✓ Scroll horizontally if cards feel cramped
✓ Use sort dropdown for different perspectives

## FAQ - Common Questions

**Q: What if I don't fill in any filters?**
A: You'll see all alumni, sorted by name. Great for browsing!

**Q: Can I combine multiple filters?**
A: Yes! Combine as many as you want. Results must match ALL criteria.

**Q: Why aren't my search suggestions showing?**
A: Type at least 2 characters. JavaScript must be enabled.

**Q: If I change the sort, do I lose my search?**
A: No! All filters are preserved when you change the sort order.

**Q: Can teachers/staff use this search?**
A: Currently designed for students. Contact dev team for other user types.

**Q: How often is the alumni data updated?**
A: Check with your database administrator for update frequency.

**Q: Can I export search results?**
A: Not yet, but it's on the roadmap for future versions!

**Q: What if the alumni I want isn't found?**
A: They might not have a profile, still have old data, or use a different name. Request they update their profile.
